# Rebelia
Pizza site
Made by Piskun Valeriy,Skubenyk Yegor,Semenenko Mykhailo
Ребелія - це сайт, де ви можете обрати собі піцу яку б хотіли замовити та вибрати куди її вам доставити. Все дуже зручно: вибір вам допоможуть зробити зображення піци, її опис та перелік складових. 
Нижче представлена структура головної сторінки сайту,ви потрапляєте в меню яким можна користуватись:
```html
<header>
    <div class="logo">
        <img src="pizza_logo.png" alt="Rebelia Pizza Logo">
        <h1>Rebelia</h1>
    </div>
    <nav class="top-menu">
        <ul>
            <li><a href="#catalog">Каталог</a></li>
            <li><a href="#cart">Кошик</a></li>
            <li><a href="#contacts">Контакти</a></li>
        </ul>
    </nav>
</header>
```
Сайт підтримує пристрої з різною роздільною здатністю екрану. Це реалізовано завдяки css та @media. Продукт вже має презентабельний вигляд.
Для прикладу було додано тимчасовий елемент в кошику.

За допомогою JS було реалізовано обирання піц до кошика з каталогу, виведення дати та часу в головному footer, в інших footer-ах при наведенні на поля навігації вони плавно змінюють колір.
Додано збереження данних на сорінці за допомогою localStorage. Також details з html був замінений на власний acardion з використанням JS.
На сторінці Contacts при натисканні на стрілки вгору та вниз змінюється розмір шрифту тексту(в основному блоці сторінки). При 0 кількості піц, її панелька пропадає. На всіх сторінках інплементована темна тема. 
Валідація форми була перенесена в JS. Надіслана форма відправляється в консоль.
